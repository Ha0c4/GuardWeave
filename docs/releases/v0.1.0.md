# GuardWeave v0.1.0

Initial public release.

## Highlights

- Risk-adaptive prompt-injection defense pipeline for hosted APIs and local models
- Heuristic-only mode with Python standard library only
- Local and remote judge support for risk scoring, output checks, and regex generation
- Local classifier judge training and evaluation helpers
- Bilingual documentation and benchmark comparison reports

## Included in this release

- Python package metadata for `guardweave`
- CLI commands for `inspect`, `chat`, `train-judge`, and `eval-judge`
- Example integrations for OpenAI-compatible APIs and local Hugging Face models
- GitHub-ready community files, issue templates, and CI workflows
